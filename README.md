<p align="center">API for Water Consumption Management</p>

## Description

This is an API to manage and monitor water consumption. The project was developed using the NestJS framework.

## Project configuration

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
